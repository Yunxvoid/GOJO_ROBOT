<h2 align="center"> 
   ⇝ Install Locally Or On A VPS ⇜
</h2>

```console
$ sudo apt upgrade
$ sudo apt install python3-pip
$ git clone https://github.com/yunxvoid/GOOJO_ROBOT
$ cd <Repo Name>
$ pip3 install -U -r requirements.txt
$ cd Himawari
$ cp config.py
```
 
<h3 align="center"> 
    Edit <b>config.py</b> with your own values
</h3>

<h2 align="center"> 
   ⇝ Run Directly ⇜
</h2>

```console

```
